問題



n個のデータをソートすることを実現する，以下のことを行うプログラムを作成してください．



n個の要素で構成される int 型の配列と，「昇順」（1），「降順」（-1）を表す int 型の値を引数として受け取り，配列内の要素を昇順または降順にソートするメソッドを作成する．

要素数 nの int 型の配列は，Random クラスを利用して疑似乱数を発生させることで生成するものとし，乱数を生成する際の種の値と要素数 n(n≦100) を事前に与えることで配列を生成するものとする．

コマンドライン入力に，乱数生成の際の種の値，データ数 n(n≦100)，ソートのモード（昇順／降順）をこの順に入力すると，要素数 n の乱数による配列を生成し，作成したソートのメソッドを使用して，モード（昇順／降順）に応じて n 個のデータをソートする． ソートした結果を，標準出力に出力する．

このプログラムを作成する際は，以下の要件を満たすように作成してください：

　【メソッド】

配列内のデータを昇順または降順にソートするメソッドは，以下の仕様とする：
メソッド名：sort

引数：
ソートのモード（昇順／降順） - int 型（「1」で昇順，「-1」で降順とする）対象の配列 - int 型の配列

戻り値：なし

内容：
引数として受け取った配列内のデータを，モードに指定した値に従って，昇順または降順にソートする．
ソートの方法については，講義内で示した「バブルソート」を使用する．

sort メソッド以外にも，必要に応じてメソッドを作成してよい．

　【入力】
プログラムの実行に必要なデータは，コマンドライン入力に入力する．
入力するデータは，以下に示すデータをこの順に，空白1文字を区切りとして入力する：
乱数生成の際の種の値
ソートするデータ数 n(n≦100)
ソートのモード（「1」または「-1」）：「1」で昇順，「-1」で降順

　【出力】
標準出力への出力は，ソートした n 個の整数値を1行に1つずつ出力する．
出力の際は，すべて半角数字を使用し，各整数値は左詰めで出力する．
行の先頭には空白を入れず，行の左端から出力する．

　【その他】
要素数 n(n≦100) の乱数による int 型の配列の生成は，以下の要件に従う：
乱数の生成には Random クラスを使用する．
コマンドライン入力に入力した乱数生成の際の種の値が，例えば int 型の変数 seed に格納されている場合，乱数生成の系列を，例えば　　Random rand = new Random(seed);として決定する． この場合，以降では Random クラスの変数 rand を通して乱数を発生させる．

宣言した Random クラスの変数を通して，0 ～ 999 の乱数を発生させる． 上のように変数 rand を宣言した場合，次の式で乱数を発生させる：
　　rand.nextInt(1000)

上記による乱数の発生を n回繰り返し，発生した乱数を配列の先頭要素から順に代入する．

【注意事項】

この課題のプログラムを作成する際は，配列内のデータを昇順または降順にソートする sort メソッドを，上記の要件を満たすメソッドとして作成してください．
